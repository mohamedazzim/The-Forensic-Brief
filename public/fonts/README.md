# Self-Hosted Fonts

Place woff2 font files in this directory:

- **Lora** (body text): `Lora-Regular.woff2`, `Lora-Italic.woff2`, `Lora-Medium.woff2`
- **Playfair Display** (headings): `PlayfairDisplay-Regular.woff2`, `PlayfairDisplay-SemiBold.woff2`, `PlayfairDisplay-Bold.woff2`

Download from Google Fonts: https://fonts.google.com/
Select: Lora (400, 500, 400i) and Playfair Display (400, 600, 700)

Once files are added, update typography.css to use @font-face declarations.
