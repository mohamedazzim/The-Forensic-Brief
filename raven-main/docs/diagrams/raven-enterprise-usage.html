Honest scope — what Raven is not
❌ Not a replacement for DB-level access controls — it scans code, DB enforces
❌ Not a network firewall — it detects changes in code, MDM/proxy enforces
❌ Not an LLM security product — Claude is guidance, not the security boundary
❌ Not a SIEM — it's a dev discipline layer, not a full security operations platform
